# Real-World: Togo CGI 2025

This walkthrough demonstrates Run-IQ applied to the **Code Général des Impôts (CGI) du Togo 2025** — the actual Togolese tax code. It shows how legal text becomes parameterized rules.

## Overview

The Togolese tax system includes:

| Tax | Model | Rate/Brackets |
|---|---|---|
| IRPP (Income Tax) | `PROGRESSIVE_BRACKET` | 0%, 10%, 15%, 25%, 35% |
| TVA (Value-Added Tax) | `FLAT_RATE` | 18% |
| IS (Corporate Tax) | `FLAT_RATE` | 27% |
| IMF (Minimum Tax) | `MINIMUM_TAX` | 1% with floor 500,000 XOF |
| CNSS (Social Security) | `COMPOSITE` | Employee 3.6% + Employer 17.5% |

### Calculation Example

For a taxable income of **7,500,000 XOF**:

| Bracket | Range | Taxable | Rate | Tax |
|---|---|---|---|---|
| 1 | 0 – 900,000 | 900,000 | 0% | 0 |
| 2 | 900,000 – 4,000,000 | 3,100,000 | 10% | 310,000 |
| 3 | 4,000,000 – 6,000,000 | 2,000,000 | 15% | 300,000 |
| 4 | 6,000,000 – 10,000,000 | 1,500,000 | 25% | 375,000 |
| 5 | 10,000,000+ | 0 | 35% | 0 |
| **Total** | | | | **985,000 XOF** |

For grossSalary 2,500,000:
- Employee: 2,500,000 × 3.6% = 90,000
- Employer: 2,500,000 × 17.5% = 437,500
- Injury: 2,500,000 × 2.5% = 62,500
- **Total CNSS: 590,000 XOF**
